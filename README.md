# Travel_Website-Travellaro
This project consists of a fully responsive multi-page website's front-end design which is created using HTML, CSS and JavaScript. It is a travel website which consistes of different destinations visited (blog section), travel and tourism tips, restaurants and resorts in different places within US and a contact page to get assitance in your Trip. The contact form will send an email to myself with all the details and message regarding which they need assistance. This is adaptable to both mobile and laptop screens and can be used in any device.
